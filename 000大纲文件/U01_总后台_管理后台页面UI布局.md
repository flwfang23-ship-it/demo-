# 总后台：管理后台页面 UI 布局

> 总后台是 SaaS 平台运营方使用的超级管理后台，覆盖客户管理、活动配置（联盟+第三方）、推广位查阅、H5模板管理、组件库管理、系统设置等功能。
> 本文档按 U 模板结构编写，每个后台页面标注对应页面编号，方便前端开发对应路由。

---

## 1. 管理后台页面整体树状图

```
SaaS营销平台 - 总后台（🏢）
 │
 ├── 🏠 首页大盘
 │   └── 数据看板（关键指标卡片 + 趋势图）
 │
 ├── 👥 用户管理
 │   ├── 客户列表（G01-g）
 │   │   └── 客户详情抽屉（G01-h）

 │
 ├── 🎯 活动管理
 │   ├── 联盟活动
 │   │   ├── 活动列表（B04-A / G02）
 │   │   │   ├── 行内操作：默认取链活动开关（G05-C）
 │   │   │   ├── 行内操作：口令自动更新标记（G05-A）
 │   │   │   └── 行内操作：关联H5模板下拉（G05-B）

 │       ├── 分组列表（B04 / G04）
 │       │   └── 分组新增/编辑弹窗（D-B04-A）
```

## 3. 后台页面布局通用规范

| 规范项 | 说明 |
| --- | --- |
| **整体布局** | 左侧固定宽度导航栏（240px）+ 右侧自适应内容区（最小宽度1200px） |
| **导航高亮** | 当前所在的一级菜单高亮，展开的二级菜单高亮 |
| **面包屑** | 二级及更深页面顶部显示面包屑导航 |
| **返回按钮** | 表单页/详情页左上角提供"← 返回列表" |
| **保存/取消** | 表单页底部固定或跟在内容末尾，保存和取消按钮并排 |
| **空状态** | 列表无数据时展示空状态插画 + 引导文案 |
| **加载态** | 列表和详情页加载中展示骨架屏或 loading |
| **错误态** | 接口异常展示错误提示 + 重试按钮 |
| **权限控制** | 超级管理员可管理所有功能，运营角色按分配的权限展示菜单 |
| **平台参数脱敏** | AppSecret 类字段默认 `****************`展示，hover/点击👁可见明文 |
| **异步导出** | 大数据量导出为异步任务，Toast "导出任务已创建"后到导出记录下载 |
