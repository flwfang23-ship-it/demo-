# 客户端后台：管理后台页面 UI 布局

> 客户端后台是每个商户独立使用的管理后台。整体布局为左侧深色侧边栏 + 右侧内容区（顶栏面包屑 + 可滚动内容区）。
> 页面通过 XHR 加载 `pages/{page}.html` 注入 `#pageContent`。

---

## 1. 管理后台页面整体树状图（实际菜单结构）

```
SaaS营销平台 - 客户端后台
 │
 ├── 🏠 工作台
 │   ├── 首页概览          → dashboard.html
 │   └── 新人指引          → onboarding.html
 │
 ├── 📢 推广管理
 │   ├── 活动清单          → activity-list.html（普通活动 + 第三方活动双Tab）
 │   ├── 推广位管理        → promotion-list.html（分组Tab + 列表 + CRUD）
 │   └── 推广分组          → promotion-group-list.html（分组独立管理页）
 │
 ├── 📱 小程序
 │   ├── 小程序管理（有数据）→ miniprogram-list.html
 │   ├── 小程序管理（空状态）→ miniprogram-list-empty.html
 │   └── 权益页配置        → benefit-config.html（手机模拟器 + 可视化编辑器）
 │
 ├── 📊 数据报表
 │   └── 效果报表          → reports-list.html（日报列表 + 详情抽屉）
 │
 ├── ⚙️ 账户设置
 │   ├── 账号信息          → account.html (tab=user)
 │   ├── 企业信息          → account.html (tab=info)
 │   ├── 修改密码          → account.html (tab=pwd)

 └── 🔑 登录注册
     └── G01 用户登录注册   → login.html（独立登录页，不经过 index.html 壳）
```

| 规范项 | 说明 |
| --- | --- |
| **侧边栏** | 240px 固定宽度，深色背景（`#001529`），支持收起/展开 |
| **菜单结构** | 分组标题 + 子项全平铺，无折叠/展开交互，当前页菜单项高亮 + 左侧白色竖条指示 |
| **顶栏** | 56px 高度，白色背景，底部阴影；左侧面包屑，右侧通知图标 |
| **内容区** | flex:1，浅灰背景（`#F5F6FA`），overflow-y:auto |
| **最小宽度** | 1200px，PC 端管理后台 |

## 4. 后台页面布局通用规范

| 规范项 | 说明 |
| --- | --- |
| **整体布局** | 左侧 240px 深色侧边栏 + 右侧 flex:1 内容区 |
| **顶栏** | 56px 白色顶栏，左侧面包屑，右侧通知图标 |
| **菜单高亮** | 当前页菜单项 `.active`：蓝色背景 + 左侧白色竖条指示器 |
| **面包屑** | 顶栏显示 🏠 首页 / 一级菜单 / 二级菜单 |
| **页面加载** | XHR 加载 pages/*.html → 分离 script 标签 → 注入 HTML → 执行脚本 → 调用 onPageLoaded() |
| **弹窗** | `.modal-overlay` 蒙层 + `.modal-box` 弹窗，支持带动画的打开/关闭 |
| **抽屉** | `.drawer-overlay` + `.drawer`，从右侧滑出 |
| **Toast** | `window.showToast(message, type)` 全局提示 |
| **空状态** | `.empty-state` + `.empty-state-icon` + `.empty-state-text` |
| **加载态** | `.loading-spinner` 旋转动画 |
| **分页** | `.pagination` + `.page-btn` |
| **确认弹窗** | `showConfirm(title, msg, onConfirm)` 通用确认 |
