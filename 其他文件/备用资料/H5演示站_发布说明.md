# H5 演示站 — 发布说明

## 线上地址

**https://flwfang23-ship-it.github.io/demo-**

## 本地预览

```bash
npx serve h5-demo -l 3000
# 浏览器打开 http://localhost:3000
```

## 部署方式

GitHub Pages + GitHub Actions，推送到 `main` 分支自动部署。

- 仓库：https://github.com/flwfang23-ship-it/demo-
- 部署配置：`.github/workflows/deploy.yml`（将 `h5-demo/` 部署到 Pages）
- Pages 设置：Source = "GitHub Actions"

## 维护备注

- 本地服务器 killed: `taskkill //F //IM node.exe`
- 纯静态项目，无构建工具
- 修改代码后 `git push` 即可，约 30 秒后线上生效
