# H5 Demo 业务模块索引

> **用途**：AI 阅读本文件后，能快速定位每个 H5 演示模块对应的产品功能、交互流程和业务逻辑，便于撰写产品功能文档。

## 1. 系统概览

**产品定位**：小程序 SaaS 营销平台 —— 客户管理后台。面向商户（如餐饮/零售商家），提供从"注册小程序→配置推广位→获取推广链接→查看数据报表"的一站式推广管理能力。

**用户角色**：商户运营人员（单一角色，无多角色权限区分）。

**页面架构**：PC 端管理后台，左侧 240px 侧边栏导航 + 右侧内容区（顶栏 56px + 页面内容），最小宽度 1200px。页面通过 XHR 加载 HTML 片段注入内容区，无路由库。

**菜单结构（5 个一级模块 + 演示专用）**：

| 一级菜单 | 二级页面 | 对应模块 |
|----------|---------|---------|

| 推广管理 | 推广位管理 | C03 推广位列表 |
| 推广管理 | 推广分组 | 推广分组管理 |

| 账户设置 | 企业信息 | C06 账号设置 |
| 账户设置 | 修改密码 | C06 账号设置 |

## 3. C01 首页推广工作台

### 3.1 业务目的

商户登录后的首页，展示收入概览、资金余额和业务推广核心操作入口。目标是让商户一眼看到"昨天赚了多少"和"能提现多少钱"，并快速进入推广取链流程。

### 3.2 页面布局

上下两排布局：
- **上排（左右分栏 2:1）**：
  - 左：昨日收入卡片（模块②）
  - 右：资金余额卡片（模块③）
- **下排（全宽）**：取链模块（模块①）—— 按供应商 Tab 分组的活动卡片网格

### 3.3 子功能详解

#### 3.3.1 新人引导横幅

| 属性 | 说明 |
|------|------|
| 触发条件 | 商户未完成全部新手引导步骤时显示 |
| 位置 | 页面最顶部，所有卡片上方 |
| 内容 | 图标 + 引导文案 + 进度文字 + "去完成"按钮 |
| 点击行为 | 跳转到新人引导页（onboarding.html） |
| 完成条件 | 4 个步骤全部 done 后，横幅消失 |

#### 3.3.2 昨日收入卡片

| 属性 | 说明 |
|------|------|
| 数据 | 昨日预估收入（¥ 金额）+ 环比变化率（↑/↓ 百分比）+ 累计收入 |
| 交互 | 点击卡片跳转到 C05 效果报表 |
| 空状态 | 无数据时显示 "暂无昨日数据" 占位 |
| 动画 | 金额数字滚动动画（animateNumber） |

#### 3.3.3 资金余额卡片

| 属性 | 说明 |
|------|------|
| 样式 | 蓝色渐变背景，区别于其他卡片 |
| 数据 | 可用余额 + 提现中金额 + 已结算金额 |
| 主要操作 | "提现" 按钮 → 打开提现弹窗 |
| **提现弹窗** | 输入提现金额（校验：>=1元 且 <=可用余额）+ 确认提现 |
| **未绑定微信** | 不展示提现按钮，改为 "绑定微信" 引导按钮 + 提示文案 |
| 提现成功 | Toast 提示 + 更新余额数据 |

#### 3.3.4 取链模块（活动推广核心区）

| 属性 | 说明 |
|------|------|
| 结构 | 顶部供应商 Tab 栏 + 下方活动卡片网格（3 列，窄屏 2 列） |
| Tab 项 | 全部 / 美团联盟 / 饿了么联盟 / 京东 / 打车 / 其他 |
| 卡片内容 | 活动图(16:9) + 活动名称 + 副标题 + 供应商标签 + 佣金 + 有效期 |
| 卡片操作 | "立即推广" 按钮 → 打开推广链接抽屉 |
| **推广链接抽屉** | 详见 3.4 节 |
| 空状态 | "暂无推广活动" 占位图 |

### 3.4 推广链接抽屉（Drawer）

这是整个系统的核心取链交互，C01 和活动清单页都会调用。

| 区域 | 内容 |
|------|------|
| 标题栏 | 活动名称 |
| 信息区 | 黄色背景提示条，显示活动名称 + 活动描述 |
| 推广位选择 | 单选列表，列出所有推广位（含推广位名称 + 所属分组） |
| 推广渠道 | 小程序推广 / H5 推广（单选切换），不同渠道显示不同配置项 |
| **小程序渠道** | 展示小程序列表（名称 + AppID + 路径 + 部署状态 + 二维码占位） |
| **H5 渠道** | H5 推广链接输入/生成区（含"生成链接"按钮 + 复制按钮） |
| 自定义链接 | "新增自定义链接" 入口，可添加多个自定义链接（名称+URL） |
| 底部操作 | "生成推广链接" 主按钮 → 下方显示生成的短链接 + 复制 |
| **取链清单全屏** | 抽屉内"查看取链清单"入口，打开全屏弹窗，展示该推广位下全部已生成链接（表格 + 展开详情 + 二维码 + 删除） |

**渠道切换的状态流转**：
- 选"小程序推广" → 展示小程序卡片列表（含部署状态、二维码）
- 选"H5 推广" → 展示路径输入框 + 生成按钮 → 生成后展示结果
- 两种渠道各自独立生成链接，互不干扰

**第三方活动的特殊处理**：
- 第三方活动（花小猪/滴滴等）在抽屉中额外展示"推广路径配置"区域
- 已配置路径 → 正常取链
- 未配置路径 → 引导去配置（跳转活动详情页）

## 4. 新人引导流程页

### 4.1 业务目的

帮助新注册商户按步骤完成账号配置，降低上手门槛。通过进度可视化 + 分步引导，让商户明确知道"还要做什么才能开始赚钱"。

### 4.2 页面布局

- **顶部**：标题 + 描述 + 进度条（已完成 N/4，百分比）
- **主体（左右分栏）**：
  - 左侧：4 步流程列表（垂直时间线样式）
  - 右侧：当前步骤的详情面板（说明 + 准备清单 + 操作按钮）

### 4.3 4 个步骤

| 步骤 | 名称 | 默认状态 | 说明 |
|------|------|---------|------|
| 1 | 注册账号 | done（已完成） | 自动生成默认推广位 |
| 2 | 上传营业执照 | current（进行中） | 财务结算和微信小程序注册的前置条件 |
| 3 | 注册微信小程序 | todo（未完成） | 法人扫脸快速注册 |
| 4 | 开始推广变现 | todo（未完成） | 全部配置完成后即可取链推广 |

### 4.4 步骤状态

| 状态 | 视觉 | 说明 |
|------|------|------|
| done | 蓝色渐变圆形 ✓ + 蓝色标签"已完成" | 已完成的步骤 |
| current | 白色圆形蓝色边框 + 橙色标签"进行中" + 外圈光晕 | 当前需操作的步骤 |
| todo | 灰色圆形 🔒 + 灰色标签"未完成" | 尚未解锁的步骤 |

### 4.5 关键交互

- 点击左侧步骤 → 右侧面板切换显示对应步骤的详情
- "去上传营业执照" 主按钮 → Toast 提示"功能开发中"（Demo 占位）
- "稍后再说" → 返回首页

## 5. 活动清单

### 5.1 业务目的

商户浏览所有可推广的活动池，包括平台提供的普通活动（美团/饿了么）和第三方活动（京东/打车/其他）。支持搜索过滤和取链操作。

### 5.2 页面结构

- **顶部 Tab 栏**：全部 / 美团联盟 / 饿了么联盟 / 京东（外部链接）/ 打车（外部链接）/ 其他（第三方链接）
- **搜索栏**：关键词搜索 + 查询按钮
- **"其他" Tab 特殊**：显示"+ 新增活动"按钮，用于添加第三方活动
- **分区标题**：当选择"全部"Tab 时，第三方活动前显示分区标题"第三方活动"

### 5.3 普通活动列表项

| 字段 | 说明 |
|------|------|
| Banner 图 | 160×90 缩略图 |
| 活动名称 | + 状态标签（进行中/已结束）+ 供应商标签（美团/饿了么） |
| 副标题 | 可选 |
| 会场ID | 可选 |
| 时间 | 开始日期 / 截止日期 |
| 佣金 | 大号橙色数字（或"--"） |
| 操作 | "立即推广" 按钮 → 打开取链抽屉（同 C01 的抽屉） |
| 操作 | "会场推广规则 >" 链接 → Toast 提示 |

### 5.4 第三方活动列表项

| 字段 | 说明 |
|------|------|
| 平台标签 | 蓝色标签（如"花小猪"） |
| 活动名称 | + "使用中"标签（如果在权益页有配置路径） |
| AppID | 灰色小字 |
| 操作 | 点击整行 → 打开"配置推广位路径"抽屉 |
| 操作 | "删除"按钮（预设活动无此按钮） |

### 5.5 第三方活动路径配置抽屉

| 区域 | 内容 |
|------|------|
| 信息区 | 平台名称 + 活动名称 + AppID（可复制） |
| 配置卡片 | 统一路径开关 + 两种模式切换 |
| **逐推广位模式**（默认） | 表格列出所有推广位 + 每行一个路径输入框 + 保存按钮 + "一键保存全部" |
| **统一路径模式** | 一个输入框配置所有推广位使用相同路径 |
| 切换行为 | 开启统一路径 → 从已有逐推广位路径中取第一个非空值预填；关闭统一路径 → 用统一路径值回填所有推广位 |

### 5.6 删除第三方活动

| 场景 | 行为 |
|------|------|
| 活动在权益页无配置 | 二次确认弹窗 → 确认后删除 |
| 活动在权益页有配置 | 弹窗提示"无法删除"，列出已配置的权益页，引导"前往权益页配置"先取消关联 |

### 5.7 新增第三方活动弹窗

| 字段 | 校验 |
|------|------|
| 平台 | 必填，如"花小猪" |
| 活动名称 | 必填，同平台下不可重名 |
| AppID | 必填 |

## 6. C03 推广位管理

### 6.1 业务目的

管理商户的推广位（广告位），是推广体系的核心组织单元。推广位可以理解为"在哪个位置推广"，关联到小程序页面路径。推广位按分组归类管理。

### 6.2 推广位列表页

| 区域 | 内容 |
|------|------|
| 分组 Tab 栏 | 顶部横向 Tab，每个分组一个 Tab，包含"管理分组"入口 |
| 筛选栏 | 推广位名称搜索 + 类型筛选 + 推广页筛选 + 查询/重置 |
| **默认推广位** | 系统自动创建，行背景黄色高亮（`promo-row-default`），不可删除 |
| 表格列 | 推广位名称、推广位ID、分组、类型(小程序/H5/小程序+H5)、小程序AppID、小程序路径、H5推广路径、缺失H5状态、权益页状态、创建时间、操作 |
| 操作 | 编辑 / 删除 / 路径编辑（推广位内第三方活动的路径） |

### 6.3 推广位表单弹窗

| 字段 | 说明 |
|------|------|
| 推广位名称 | 必填 |
| 所属分组 | 下拉选择 + "新建分组"内嵌入口 |
| 类型 | 小程序 / H5 / 小程序+H5 |
| AppID | 类型含小程序时显示 |
| 小程序路径 | 类型含小程序时显示 |
| H5 路径 | 类型含 H5 时显示 |

### 6.4 内嵌新建分组

在推广位表单弹窗内，分组下拉框旁有"新建"链接 → 展开内嵌输入框 → 输入分组名称 → 确认后创建并自动选中 → 或取消收起

### 6.5 推广分组管理页（独立页签）

| 功能 | 说明 |
|------|------|
| 列表 | 分组名称、描述、包含推广位数、创建时间、操作 |
| 默认分组 | 系统默认，不可删除（红色"默认"标签） |
| 新增/编辑 | 弹窗表单：名称 + 描述 |
| **删除分组** | 如果分组下有推广位 → 弹窗提示"请先将推广位移到其他分组"（含迁移下拉选择+确认迁移按钮）；如果分组下无推广位 → 直接删除 |

### 6.6 推广位路径编辑弹窗

打开推广位的路径编辑弹窗，按第三方活动维度展示每个第三方活动的路径配置状态：
- 已配置（绿色）→ 显示路径值，可直接编辑
- 统一路径模式（蓝色）→ 显示统一路径值，不可编辑（引导去活动详情页修改）
- 未配置（橙色）→ 输入框为空，高亮提示

### 6.7 缺失 H5 状态

每个推广位的列表行中展示"缺失H5"状态标签：
- 绿色 "正常"：所有必需路径已配置
- 橙色 "N个活动待配置"：有 N 个活动缺少 H5 推广路径

## 7. C02 小程序管理

### 7.1 业务目的

管理商户的微信小程序，包括注册、部署状态追踪、订阅开关。一个商户可以有多个小程序，每个小程序可关联到不同推广位。

### 7.2 页面状态

| 状态 | 页面 | 触发条件 |
|------|------|---------|
| 空状态 | miniprogram-list-empty.html | 商户尚未注册任何小程序 |
| 有数据态 | miniprogram-list.html | 已注册至少一个小程序 |

### 7.3 列表页功能（有数据态）

| 功能 | 说明 |
|------|------|
| 搜索 | 小程序名称/AppID 搜索 |
| 筛选 | 按部署状态筛选（已上线/审核中/部署中） |
| 批量操作 | 批量订阅 / 批量取消订阅 |
| 表格列 | 小程序名称、AppID、部署状态(彩色标签)、订阅状态(开关)、到期时间、操作 |
| 注册按钮 | 打开注册表单弹窗 |
| 编辑 | 打开编辑表单弹窗（名称 + AppID） |
| 详情 | 打开详情弹窗（展示部署进度时间线 + 基本信息） |
| 订阅开关 | 行内 Switch 开关，点击切换订阅状态 → Toast 确认 |
| 订阅操作记录 | 弹窗展示订阅操作日志列表（时间+操作+操作人） |
| 分页 | 底部分页组件 |

### 7.4 部署状态

| 状态 | 标签颜色 | 说明 |
|------|---------|------|
| 已上线 | 绿色 | 小程序已审核通过并上线 |
| 审核中 | 橙色 | 小程序提交审核中 |
| 部署中 | 蓝色 | 小程序部署流程进行中 |

### 7.5 订阅机制

- 订阅 = 商户同意接收该小程序的推广数据推送
- 开启订阅 → 该小程序的推广数据进入 C05 日报统计
- 关闭订阅 → 停止数据推送
- 订阅开关操作记录在操作日志中

## 8. C04 权益页配置（核心编辑器）

### 8.1 业务目的

商户可视化配置用户端（手机端）权益页的展示内容。这是整个系统最复杂的模块，本质是一个"拖拽式页面搭建器"，让商户自己编排手机端用户看到的优惠活动页面。

### 8.2 页面布局

三栏布局：
- **左侧**：组件库（活动卡片列表）
- **中间**：手机模拟器（420px 宽，模拟真实小程序权益页）
- **右侧**：推广位配置面板（表格 + 管理入口）
- **顶部**：推广位 Tab 栏 + "管理"入口

### 8.3 手机模拟器结构

```
┌─────────────────────┐
│  手机导航菜单（左侧）  │ ← 可编辑的分类菜单（增删改）
│  ├ 推荐              │
│  ├ 外卖              │
│  ├ 打车              │
│  └ ...               │
├─────────────────────┤
│  内容画布（右侧）     │ ← 拖拽放置区
│  ├ 模块标题栏         │ ← 可编辑标题+Logo
│  ├ 活动卡片1（大卡）  │ ← 可拖拽排序，可编辑样式/选择活动
```

## 13. 全局交互模式

### 13.1 弹窗体系

| 类型 | 使用场景 | 关闭方式 |
|------|---------|---------|
| modal-overlay | 表单弹窗（新增/编辑） | 点关闭按钮、点蒙层、ESC |
| drawer-overlay | 侧边抽屉（取链、路径配置、报表详情） | 点关闭按钮、点蒙层 |
| confirm-overlay | 二次确认弹窗（删除等危险操作） | 点取消/确认按钮、点蒙层 |
| Toast | 操作结果反馈 | 自动消失 |
| 全屏弹窗 | 取链清单 | 点关闭按钮 |

### 13.2 空状态处理

| 页面 | 空状态表现 |
|------|-----------|
| 小程序管理 | 独立空状态页，引导"注册第一个小程序" |
| 活动清单 | "暂无活动"占位图标 + 文字 |
| 推广位管理 | "暂无推广位"空表格 |
| 取链模块 | "暂无推广活动"网格内居中占位 |
| 首页收入 | "暂无昨日数据"文字占位 |

### 13.3 加载状态

- 页面切换：居中 Loading Spinner（带放大动画）
- 数据提交：按钮 Loading 态（disabled + 文字变为"保存中..."）

### 13.4 Toast 反馈

- 成功：绿色（success）
- 错误：红色（error）
- 提示：蓝色（info）
- 统一从顶部居中弹出，3 秒自动消失
