PC端登录页

2.1 用户路径图 (User Flow)
                     PC端登录页
                         │
         ┌───────────────┴───────────────┐
         ▼                               ▼
   [扫码登录 Tab]                  [账号密码登录 Tab]
         │                               │
         │ 展示带参二维码                 │ 输入手机号+密码
         │                               │
         ▼                               ├── 点击"忘记密码" → 密码重置流程
   用户微信扫码                           │
         │                               ▼
         ├── 老用户(已绑定) ──────────► 校验通过 → 进入SaaS工作台
         │                               │
         └── 新用户(未关注/未绑定)        │ 校验失败 → 错误提示
                │                        │
                ▼                        │
         关注公众号                       │
                │                        │
                ▼                        │
         收到自动回复(绑定链接)            │
                │                        │
                ▼                        │
         H5绑定手机号页                   │
                │                        │
                ├── 快捷：微信授权获取手机号
                └── 常规：手动输入+短信验证码
                │                        │
                ▼                        │
         绑定成功 → 跳转H5修改密码页       │
                │                        │
                ├── 设置新密码并确认       │
                └── 或点击"暂不修改"       │
                │                        │
                ▼                        │
         PC端监听到绑定成功 → 自动刷新进入工作台
