## 项目背景

小程序SaaS营销平台 — 客户管理后台 H5 Demo。纯 HTML+CSS+JS，无构建工具。PC 端管理后台，最小宽度 1200px，左侧侧边栏 + 右侧内容区布局。

**当前状态**：代码位于 `h5-demo/` 目录。

**模块清单**：

| 编号 | 模块 | 核心页面 |
|------|------|----------|
| C01 | 首页概览（商户推广工作台） | 昨日数据 + 财务数据 + 取链模块 |
| C02 | 小程序管理 | 列表 + 注册/编辑 + 详情 + 订阅开关 |
| C03 | 推广位管理 | 列表 + 新增/编辑 + 设为默认 + 删除 |
| C04 | 权益页配置（核心） | 左右分栏 + Tab + 6坑位网格 + 活动选择器弹窗 + 推广位管理弹窗 |
| C05 | 日报 | 列表 + 详情 |
| C06 | 账号设置 | 企业信息 + 修改密码 + 操作日志 |
